# Report for F-216: кнопки edit/cancel/start только для creator/admin

**Agent:** Codex  
**Date:** 2026-05-01  
**Status:** `completed`

## Summary

В `MatchDetailPage` action-кнопки управления матчем теперь последовательно видны только создателю партии или администратору. Довёл это правило и для `in_progress` состояния: обычный участник больше не видит `Трекер раундов` и `Завершить`.

---

## Changes

- `frontend/src/pages/MatchDetailPage.tsx` — `in_progress` action-buttons обёрнуты в тот же `isCreatorOrAdmin`, который уже использовался для `planned`-кнопок.
- `frontend/tests/matches-pages.test.tsx` — добавлен regression-тест на обычного участника: в `planned` он не видит `Начать партию / Редактировать / Отменить партию`, а в `in_progress` не видит `Трекер раундов / Завершить`.
