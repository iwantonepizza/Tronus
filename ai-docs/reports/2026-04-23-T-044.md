## Summary

Added public `GET /api/v1/stats/head-to-head/?user_a=X&user_b=Y` for comparing two players. The endpoint returns pair metadata, aggregate counters, favorite factions in common games, and a list of completed sessions where both players participated.

## Decisions made

- Head-to-head is computed only from completed sessions where both users participated.
- Aggregate payload includes:
  - `games_together`
  - `wins`
  - `higher_place`
  - `favorite_factions`
  - `matches`
- `favorite_factions` are derived from common completed games only, not from each player’s global history.
- `matches` are sorted by newest first and include enough per-user data for the future H2H screen:
  - faction
  - place
  - castles
  - `is_winner`
