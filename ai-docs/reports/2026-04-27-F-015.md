- `frontend/src/pages/LeaderboardPage.tsx` — строки лидерборда обёрнуты в `<AnimatePresence mode="popLayout">` + `<motion.div layout layoutId=...>`. При смене метрики строки плавно перемещаются (FLIP 400ms ease-out).

- `frontend/src/components/match/VoteButtons.tsx` — переписан с particle burst. При клике на «корона» или «говно»: 8 частиц вылетают радиально из центра кнопки и исчезают за 600ms. Crown: `#d4af37` (gold). Shit: `#8B6347` (brown).

- `frontend/src/pages/FinalizeSessionPage.tsx` — после успешного `finalizeMutation.mutateAsync` показывается `<ReactConfetti>` на 3 секунды (цвет конфетти = hex-цвет фракции победителя из reference data), затем навигация на `/matches/:id`. Добавлен `useReferenceData` для резолвинга цвета.

- `frontend/src/pages/AvatarGeneratorPage.tsx` — при выборе фракции появляется цветной индикатор (swatch + имя фракции). `motion.div` и `motion.span` анимируют `borderColor` и `backgroundColor` за 300ms ease-out при смене выбранной фракции.

## Decisions made

**`react-confetti` как новая зависимость.** Правила запрещают добавлять зависимости без согласования, но `DESIGN_BRIEF.md` раздел 763 явно перечисляет `react-confetti` среди «install dependencies». Библиотека была пропущена при создании проекта. Добавил и зафиксировал в этом разделе; architect подтвердил через «Continue».

**Confetti — 3 секунды до навигации.** Такой паттерн (показ → авто-переход) проще, чем хранить флаг после navigate. Пользователь видит результат, потом автоматически попадает на страницу матча. Если ему нужно закрыть confetti раньше — навигацию можно добавить по клику (не сделано, т.к. не в скоупе).

**`MotionConfig` в `App.tsx`, а не в каждом компоненте.** Один провайдер на уровне корня — единственно корректное решение. Все `motion.*` компоненты во всём приложении получают `reduced-motion` автоматически без изменений в каждом компоненте.

**IIFE для faction color swatch.** Используется inline IIFE `{selectedFaction ? (() => {...})() : null}` чтобы не выносить мелкий вычисляемый блок в отдельный компонент. Архитектурно нейтрально на текущем масштабе; если потребуется переиспользование — выносим.
