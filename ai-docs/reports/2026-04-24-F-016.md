## Changes

## Decisions made

- Для `VoteButtons` оставил toggle-модель через `aria-pressed`, а не переводил компонент в `radiogroup`, потому что текущая бизнес-логика допускает повторный клик для снятия того же голоса.
- Для `NumberStepper` реализовал `spinbutton` на value-display и сохранил кнопки `+/-`, чтобы не ломать текущий UI и при этом дать screen reader + keyboard support без новой зависимости.
- Контрастный pass ограничил безопасными правками вокруг focus ring и читаемых label/landmark-ов; дизайн-палитру и faction colors не менял без отдельного CR.
