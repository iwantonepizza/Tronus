## Summary

Добавил в `RoundTrackerPage` UI для фиксации Westeros event cards: модалка с 3 или 4 dropdown по активным колодам, возможностью оставить любую колоду пустой и последовательной отправкой выбранных карт в timeline API. Довёл задачу до конца отдельным UI-тестом и полным зелёным прогоном фронта.

## Decisions made

- Так как отдельного frontend reference endpoint для event decks сейчас нет, использовал локальный read-only каталог карт, синхронизированный с backend `apps/games/event_cards.py`.
- Один submit wizard-а отправляет несколько POST-запросов подряд: по одному на каждую непустую колоду. Это сохраняет совместимость с текущим backend API `POST /timeline/event-card/`, который принимает только одну карту за запрос.
- Для кнопки открытия модалки добавил `aria-label="open-event-cards"`, чтобы тест и a11y не зависели от кодировочных проблем текстового matcher-а.
