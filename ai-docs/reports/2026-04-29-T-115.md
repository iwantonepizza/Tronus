# Report for T-115: Russian validation messages in backend

**Agent:** Codex
**Date:** 2026-04-29
**Status:** `completed`

---

## Summary

Перевёл backend validation и API error messages на русский в затронутых apps, а также переключил Django на `ru-ru`, чтобы встроенные serializer/password messages тоже приходили локализованными. Дополнительно обновил тесты под новые тексты и зафиксировал служебную migration для `Faction` validator metadata.

## Decisions made

- Переключил `LANGUAGE_CODE` на `ru-ru`, чтобы не переводить вручную встроенные Django/DRF validation messages и password validator texts по всему проекту.
- Оставил `error.code` англоязычным (`validation_error`, `invalid_credentials`, `not_found` и т.д.), как требует задача и ADR-0015; переведены только `message` и `details`.

## Deviations from task

- Добавлена `backend/apps/reference/migrations/0007_localize_hex_color_validator.py`. Формально задача была про тексты, но локализация `HEX_COLOR_VALIDATOR` меняет Django model state, поэтому без этой migration `makemigrations --check` стабильно падал. Изменения таблиц по сути нет, только metadata validator message.
