# Change Requests

Реестр change requests и значимых уточнений scope. Новые записи добавляются в начало.

Формат:

```md
## YYYY-MM-DD — CR-XXX: <название>
- status: proposed | approved | rejected | implemented
- related task: T-XXX | -
- file: ai-docs/changes/YYYY-MM-DD-CR-XXX.md
- summary: <краткое описание изменения>
```

---

(Пусто. Первая запись появится при первом изменении scope.)
